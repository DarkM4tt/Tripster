# Tripster

Tripster is a React-based website that allows users to discover and plan their next trip. It provides a user-friendly interface to explore destinations, view attractions, and save favorites for future reference. This project was created as a coding test for the Front-End Developer position at FLXlease, inc.

## Technologies Used

- React
- Tailwind CSS
- react-icons

## Installation

1. Clone the repository:

https://github.com/DarkM4tt/Tripster

2. Navigate to the project directory:

cd tripster

3. Install the dependencies:

npm install

## Usage

1. Start the development server:

npm start

2. Open your browser and visit `http://localhost:3000` to see the Tripster website.

## Folder Structure

The project follows a standard React project structure:

tripster/
├── public/
│ ├── assets
| |\_\_index.html
| |
│ └── ...
├── src/
│ ├── components/
│ │ ├── Demo.js
│ │ ├── Description.js
│ │ └── ...
│ ├── App.js
│ ├── index.js
│ └── ...
├── .gitignore
├── package.json
├── README.md
└── ...

- The `public` directory contains the main HTML file and other static assets file.
- The `src` directory contains the components and the main `App.js` file.
- The `components` directory contains reusable React components used throughout the application.
- The `App.js` file is the entry point of the application.
